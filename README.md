BrightTV Viewership Analytics Case Study

Project Overview

This project was completed as part of the BrightTV Viewership Analytics Case Study. The objective was to analyze subscriber viewing behaviour and provide business insights that would help BrightTV grow its subscription base. The analysis focused on identifying user trends, understanding the factors influencing content consumption, and recommending strategies to improve customer engagement and increase subscriptions.

Project Objectives

* Analyze user viewing behaviour and usage trends.
* Identify the factors influencing content consumption.
* Recommend content strategies for periods of low viewership.
* Suggest initiatives to increase BrightTV's subscriber base.
* Present findings through an interactive dashboard.

Tools Used

* Databricks SQL
* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* GitHub
* Google Data Studio
* Lovable
* Power BI

Dashboard Features

The dashboard includes multiple analytical views to help understand customer behaviour:

* Channel performance analysis
* Top performing TV channels
* Total watch time by channel
* Average watch duration
* Viewership by age group
* Viewership by gender
* Regional viewership analysis
* Social media engagement analysis
* Peak viewing hours
* Hourly viewing trends
* Weekday vs Weekend consumption
* Daily viewing patterns

Data Preparation

Before creating the dashboard, the dataset was cleaned and transformed using SQL in Databricks.

Key data preparation steps included:

* Cleaned missing and inconsistent values.
* Standardized demographic categories such as Gender, Race and Province.
* Grouped ages into meaningful age bands.
* Replaced blank and invalid values with "Unknown" where appropriate.
* Created calculated fields to improve reporting and visualization.

Key Business Insights

The analysis helped identify important customer viewing patterns including:

* The TV channels attracting the highest number of viewers.
* The channels generating the longest viewing durations.
* Peak hours when subscribers are most active.
* Differences in weekday and weekend viewing behaviour.
* Audience distribution across age groups and regions.
* Viewer engagement by gender and social media activity.
* Average viewing duration across different channels.
* Opportunities to increase engagement during periods of low consumption.

Business Recommendations

Based on the analysis, several recommendations were developed:

* Schedule premium and exclusive content during low-viewership periods.
* Personalize content recommendations based on viewer demographics.
* Increase targeted marketing campaigns using customer segmentation.
* Promote high-performing channels to attract new subscribers.
* Encourage user engagement through loyalty programmes and promotional offers.
* Use viewing trends to optimise programming schedules.
